# Language Selection Guide

## How to Change Your Language

### Step 1: Open Settings

1. Click on **Settings** in the navigation menu
2. Or go to `/app/settings` in your browser

### Step 2: Select Language

1. Find the **Language** section in the General settings
2. Click on the language dropdown
3. Select your preferred language from the list

### Step 3: Language Applied

Your language preference will be:
- ✅ Applied immediately to the entire UI
- ✅ Saved to your account (if logged in)
- ✅ Remembered for future visits

11. **Русский** (ru) - Russian

## Language Preference Sync

### Logged-In Users

- Your language preference is saved to your account
- It syncs across all your devices
- It's used as the default language for summary generation

### Guest Users

- Your language preference is saved in your browser
- It only applies to the current browser/device
- It's not synced across devices

## Summary Language

When generating summaries:

1. **Default:** Your UI language preference is used automatically
2. **Override:** You can select a different language in the summary form
3. **Per Summary:** Each summary can use a different language

## Right-to-Left (RTL) Languages

Some languages (like Arabic) are written right-to-left. The application automatically:
- Adjusts the layout direction
- Mirrors navigation and UI elements
- Formats text correctly

## Troubleshooting

### Language Not Changing

1. **Refresh the page** - Sometimes a refresh is needed
2. **Clear browser cache** - Old translations might be cached
3. **Check settings** - Verify the language is selected correctly

### Some Text Still in English

This means a translation is missing. Please:
1. Report the issue to support
2. Include which page/feature has the missing translation
3. We'll add the translation in the next update

### Language Resets After Logout

- Guest users: Language is stored in browser only
- Logged-in users: Language is saved to your account
- After logout, guest mode uses browser language

## Frequently Asked Questions

### Q: Can I use different languages for UI and summaries?

**A:** Yes! Your UI language and summary language are independent:
- UI language: Controls the interface language
- Summary language: Controls the language of generated summaries

### Q: Will my language preference work on mobile?

**A:** Yes, if you're logged in, your language preference syncs to mobile devices.

### Q: Can I suggest a new language?

**A:** Yes! Please contact support with your language request. We're always adding new languages.

### Q: Why are some languages missing translations?

**A:** We're continuously improving translations. Missing translations will show in English as a fallback.
